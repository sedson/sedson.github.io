2D – AfterEffects

Layer defined by (1) content and (2) transform.

Content: vector, psd, png, text, solid, pre-comp, null...

Transform:

  1. Position – X + Y
  2. Rotation – 1 value
  3. Scale – Width + Height (X + Y)
  4. Anchor Point – X + Y

3D – Blender

Object defined by (1) content and (2) transform.

Content: mesh, curve, empty, text, light, camera...

Transform:

  1. Position – X + Y + Z
  2. Rotation – X + Y + Z
  3. Scale – X + Y + Z
  4. Origin – X + Y + Z

Graded Project 1 – Class Music Video

This is going to be our first bigger – and graded – project of the semester. We are going to be building a short music video as a class.

Each student will animate exactly 3 seconds, which will give us a 40-second video. It will be good practice working as part of a team/studio and coordinating with each other to get satisfying transitions from composition to composition. And also good to practice animating to the rhythm of music.

 

Step 1 – Style Frames

The first step is to design style (key) frames. You should watch the timing video to see what part of the song you will be animating to. Then design the frame that will appear when your name appears. Think about the mood of the song at this time. Is it spacey, tense, strong, etc.

Use this palette as a jumping off point - though you can add a color or two if need be. If we keep these colors up throughout, everything will feel more consistent.

Design your style frame using simple components that you will be able to animate convincingly.

 

Format

  • Ultra-wide: 2560 × 1080 pixels
  • PSD with separate layers your peers can animate using your components.
  • Submit on Dropbox
